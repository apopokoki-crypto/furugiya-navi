# 古着ナビアプリ 計画書

## コンセプト
古着に興味はあるけど入門できていない人の「壁」を取り除く。
特に「良い品・悪い品の見分け方がわからない」という課題をAIで解決する。

## ターゲットユーザー
- 古着に興味はあるが買ったことがない初心者
- 主に東京在住（将来的に全国へ拡大）

## 主要機能

### MVP（プロトタイプ・1週間で完成目標）
1. **古着店マップ** — 東京の古着屋を地図上に表示、店舗情報を確認できる
2. **AI品質判定** — 古着の写真を撮ると状態スコアと古着の魅力説明を表示

### 将来機能
- レビュー・口コミ機能
- 全国への店舗データ拡大
- プレミアム機能（詳細な判定レポートなど）
- 広告掲載

## 収益モデル
- 広告収益（無料ユーザー向け）
- プレミアムプラン（有料・詳細機能）

## 開発スケジュール（1週間）
| 日程 | 内容 |
|---|---|
| 1〜2日目 | 店舗マップの完成度アップ |
| 3〜5日目 | AI品質判定機能の実装 |
| 6〜7日目 | デザイン調整・動作確認 |

## 技術スタック
- フロントエンド: HTML / CSS / JavaScript
- 地図: Leaflet.js + OpenStreetMap
- AI判定: Claude API（画像解析）
